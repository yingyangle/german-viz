Motivation

Our motivation was to create a website that would be helpful and intuitive to use for anyone who is interested in learning about the German language. The purpose of the website is to walk the user through background about the German language, its relevancy to English, as well as word origins and fun words that would make the user interested in learning more. After providing a background of the German language, the grammar visualizations act as an interactive and interesting way to learn some introductory grammar rules and begin recognizing patterns among German nouns to aid the learning process.


Design Rationale

To visualize the German language, we created visualizations that would show word origins, word order, and gender patterns because these are foundational topics in learning German. We chose a bubble chart for the root words chart so that color encoding would show the origin language while having each word in a draggable bubble would let the user compare words of different origins and guess the English word. For the interesting German words, our focus was on creating an engaging and interactive visualization for the user to learn new words and be more engaged with the website and learning the language. The purpose of the map was to show the number of speakers and learners in each region of the world to encourage the user to continue learning. We chose a ternary chart to show gender patterns among nouns because there are three gender categories, so the three points on the chart would effectively encode which word endings tend to fall under each category. Since there are exceptions to the rules, word endings can fall on a scale between any of the gender categories. Finally, we selected the sankey diagram to show plural patterns because the user can clearly see the most common plural endings that a singular ending would map to.


Data Processing

In order to create these visualizations, we found the data for the most common German nouns, the gender data of those nouns, the data for German origins of English words. Data was cleaned in Python to get the gender percentages, and data was also converted to json format to create the nodes and links for force charts.
